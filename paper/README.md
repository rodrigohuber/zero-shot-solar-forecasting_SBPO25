# Paper

This repository reproduces the experiments from the following peer-reviewed paper:

> **Exploring Transfer Learning Techniques for Solar Irradiation Forecast across
> Geographically Diverse Locations in Brazil Using Reanalysis Data**
> Rodrigo Huber Marques Moreira Mendes, Fernanda Araújo Baião, Reinaldo Castro Souza
> Department of Industrial Engineering, Pontifical Catholic University of Rio de Janeiro (PUC-Rio)
> *Simpósio Brasileiro de Pesquisa Operacional (SBPO) 2025.*

**Official publication:**
<https://proceedings.science/sbpo/sbpo-2025/trabalhos/exploring-transfer-learning-techniques-for-solar-irradiation-forecast-across-geo?lang=pt-br>

## Files

- [`SBPO2025_transfer-learning-solar-forecasting.md`](SBPO2025_transfer-learning-solar-forecasting.md)
  — full text of the paper, converted to Markdown (via [Docling](https://github.com/DS4SD/docling))
  for easy searching and inline reading.

> **Note:** figures in the Markdown file are embedded as inline base64 images. GitHub's
> Markdown viewer may not render `data:` images inline — for the figures as published, see
> the official publication linked above.
